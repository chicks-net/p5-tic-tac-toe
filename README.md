## example

	   |   |  
	---+---+---
	   |   |  
	---+---+---
	   |   |  

	[0] place for X: a1

	 X |   |  
	---+---+---
	   |   |  
	---+---+---
	   |   |  

	[1] place for O: b1

	 X |   |  
	---+---+---
	 O |   |  
	---+---+---
	   |   |  

	[2] place for X: a2

	 X | X |  
	---+---+---
	 O |   |  
	---+---+---
	   |   |  

	[3] place for O: b2

	 X | X |  
	---+---+---
	 O | O |  
	---+---+---
	   |   |  

	[4] place for X: a3
